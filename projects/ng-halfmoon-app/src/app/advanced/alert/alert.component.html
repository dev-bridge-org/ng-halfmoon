Alerts

Basic Usage

You can display information to the user by adding a alert to the page. Therfore you need to add the hm-alert in your template.
The hm-alert provides a input to set the appearance of the alert. Available: primary, secondary, success and danger.
To display a normal alert without any coloring just remove the appearance-input.

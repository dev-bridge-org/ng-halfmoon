Buttons

Buttons can be used in different ways and so you can use them easy in your angular-app just add the hmButton directive to it.

All appearances

There are different appearances for the buttons you can choose between one of them and style your button as you need it.
You need to add the input appearance to style your button.

Tags

There are three different approaches to use show buttons.
You can use the following tags: button, input and a.

Link

Sizing

A button can have different sizes, so you can adjust it as you need and where it is positioned.
Therefore you can add the input size with two options: sm and lg.
When your button should have the normal size simply don't add the input.


Shapes

Your buttons can have different shapes by adding the input shape.
The input has the following options: rounded and square.

Disabled

Disabling buttons is important to deny users from doing actions they should not do.
This is achievable by adding the input disabled with a boolean value to the button.
